## Architecture Docs Index

This folder contains high‑level architecture and design documents for PrivatePay.

- **`ARCHITECTURE_ANALYSIS.md`** – overall system analysis and design trade‑offs.
- **`BRIDGE_ARCHITECTURE.md`** – detailed design for all Zcash‑centric bridges (Solana, Starknet/Ztarknet, Osmosis, Miden).
- **`SYSTEM_FLOWS.md`** – end‑to‑end flow diagrams and narrative for core user journeys.
- **`STABLECOIN_DESIGN.md`** – architecture and risk model for the pZUSD stablecoin and collateral pools.
- **`IMPLEMENTATION_REVIEW.md`** – technical review of what’s implemented vs. planned.
- **`IMPLEMENTATION_STATUS.md`** – status tracking for major components and integrations.
- **`ZcashMinaBridge.md`** – design notes for the Zcash–Mina private bridge.
